用語の説明
trend_score
週間の再生数増加分 + 高評価数×10 + コメント数×50 で算出した独自の「伸びスコア」。数値が大きいほど今週勢いがある動画です。
授業化度(curriculum_fit_score)
コマンド・建築・レッドストーンなど、ニコリヒトラボの授業テーマに使いやすいタグに応じて加点したスコア。
動画案度(video_idea_fit_score)
週間の伸び幅や「作ってみた」「対決」など企画向けタイトルの傾向から、こっぴーふーチャンネルの動画企画に使いやすいかを示すスコア。
